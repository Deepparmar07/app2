# EpicBox User Guide

## Getting Started

### First Time Setup
1. **Register an Account**
   - Click "Register" on the login page
   - Enter a username (letters, numbers, underscores only)
   - Create a strong password (minimum 6 characters)
   - Click "Sign Up"
   - The first user automatically becomes an admin

2. **Login**
   - Enter your username
   - Enter your password
   - Click "Sign In"

### Understanding the Interface

#### Sidebar Navigation
- **My Files**: Main file manager dashboard
- **Recycle Bin**: View and restore deleted files
- **Shared Links**: Manage your shared files
- **Settings**: Configure your account and preferences
- **Admin Panel**: (Admin only) Manage users and system

#### Storage Indicator
- Located at the bottom of the sidebar
- Shows used storage vs. total storage
- Visual progress bar
- Updates automatically

## File Management

### Uploading Files
1. Click the "Upload" button in the top right
2. Select one or more files (max 1MB each)
3. Watch the upload progress
4. Files appear in your current folder

**Tips:**
- Images are automatically compressed
- Supported formats: Documents, Images, Videos, Audio, Archives
- Maximum file size: 1MB

### Creating Folders
1. Click "New Folder" button
2. Enter a folder name
3. Click "Create"
4. Folder appears in your current location

### Navigating Folders
- Click on a folder to open it
- Use breadcrumbs at the top to navigate back
- Click "My Files" to return to root

### Viewing Files
**Grid View:**
- Visual thumbnails
- File names below icons
- Quick overview of files

**List View:**
- Detailed file information
- File size and upload date
- Sortable columns

### Previewing Files
1. Click on a file
2. Preview opens automatically for:
   - Images (JPG, PNG, GIF)
   - Videos (MP4, AVI, MOV)
   - PDFs
3. Download button available in preview

### Downloading Files
- Click the download icon next to a file
- Or download from the preview window
- File saves to your browser's download folder

### Deleting Files
1. Click the delete icon next to a file
2. File moves to Recycle Bin
3. Restore or permanently delete from Recycle Bin

## Sharing Files

### Creating a Share Link
1. Click the share icon next to a file or folder
2. Choose permissions:
   - **Allow downloads**: Users can download the file
   - **View only**: Users can only view the file
3. Click "Generate Share Link"
4. Copy the link and share it

### Managing Shared Links
1. Go to "Shared Links" in the sidebar
2. View all your active share links
3. Copy links again if needed
4. Delete links to revoke access

### Accessing Shared Files
- Anyone with the link can access shared files
- No login required for public shares
- Permissions are enforced (view-only or download)

## Recycle Bin

### Restoring Files
1. Go to "Recycle Bin" in the sidebar
2. Find the file you want to restore
3. Click "Restore"
4. File returns to its original location

### Permanent Deletion
1. Go to "Recycle Bin"
2. Click "Delete" next to a file
3. Confirm the permanent deletion
4. File is permanently removed from storage

## Settings

### Profile Settings

#### Update Username
1. Go to Settings → Profile tab
2. Enter new username
3. Click "Update"
4. Username changes immediately

#### View Account Information
- Account type (User or Admin)
- Member since date
- Current username

#### Manage Preferences
Toggle these options on/off:
- **Auto-compress images**: Automatically compress images on upload
- **Email notifications**: Receive email updates (future feature)
- **Storage alerts**: Get notified when storage is low

### Security Settings

#### Change Password
1. Go to Settings → Security tab
2. Enter current password
3. Enter new password (min 6 characters)
4. Confirm new password
5. Click "Change Password"

**Password Requirements:**
- Minimum 6 characters
- Must match confirmation
- Cannot be empty

#### Account Security
- View security status
- See active protection methods
- Monitor account safety

#### Delete Account
1. Go to Settings → Security tab
2. Scroll to "Danger Zone"
3. Click "Delete Account"
4. Confirm deletion (feature in development)

### Appearance Settings

#### Switch Theme
**Method 1: Settings Page**
1. Go to Settings → Appearance tab
2. Toggle "Dark Mode" switch
3. Or click on theme preview cards

**Method 2: Sidebar**
- Click the sun/moon icon at the bottom of sidebar
- Theme switches instantly

**Available Themes:**
- **Light**: Clean white background, dark text
- **Dark**: Dark background, light text

### Storage Settings

#### View Storage Usage
1. Go to Settings → Storage tab
2. See detailed storage information:
   - Used storage amount
   - Total storage limit
   - Usage percentage
   - Visual progress bar

#### Storage Alerts
- Automatic warning at 80% usage
- Displayed in storage settings
- Shown in sidebar storage indicator

#### Storage Statistics
- Total files count (coming soon)
- Total folders count (coming soon)
- Storage breakdown by type (coming soon)

## Admin Features

### Accessing Admin Panel
1. Click "Admin Panel" in the sidebar (admin only)
2. View all system users

### Managing Users
**View User Information:**
- Username
- Account role (User/Admin)
- Storage usage
- Join date

**Change User Roles:**
1. Find the user in the list
2. Use the dropdown to select new role
3. Choose "User" or "Admin"
4. Role updates immediately

**Monitor Storage:**
- See each user's storage usage
- View storage limits
- Track system-wide usage

## Tips & Best Practices

### File Organization
- Create folders for different projects
- Use descriptive file names
- Regularly clean up unused files
- Check Recycle Bin periodically

### Storage Management
- Monitor your storage usage regularly
- Delete unnecessary files
- Empty Recycle Bin to free up space
- Compress large images before upload

### Security
- Use a strong password
- Change password regularly
- Don't share your account credentials
- Only share files with trusted people
- Delete share links when no longer needed

### Sharing
- Set appropriate permissions (view vs. download)
- Delete share links after use
- Monitor active share links
- Use descriptive names for shared items

## Keyboard Shortcuts

### Navigation
- Click breadcrumbs to navigate folders
- Use browser back button to go back

### File Operations
- Click to select files
- Double-click to open/preview
- Right-click for context menu (future feature)

## Troubleshooting

### Upload Issues
**File too large:**
- Maximum file size is 1MB
- Compress files before uploading
- Split large files into smaller parts

**Upload failed:**
- Check your internet connection
- Verify file format is supported
- Try uploading again

### Storage Issues
**Storage full:**
- Delete unnecessary files
- Empty Recycle Bin
- Contact admin for storage increase

### Login Issues
**Forgot password:**
- Contact administrator for password reset
- Create a new account if needed

**Cannot login:**
- Check username spelling
- Verify password is correct
- Clear browser cache and try again
